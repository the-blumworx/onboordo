# Security Policy

## Supported Versions

Currently, only the latest release is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Onboordo, please DO NOT open a public issue.

Instead, please send an email to security@onboordo.com (or use GitHub Security Advisories if enabled). We will treat your report with the utmost priority and respond within 48 hours.

We kindly ask that you wait for a patch to be published before disclosing the vulnerability publicly.
